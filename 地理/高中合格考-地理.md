# 高中合格考知识点复习 —— 地理

## 主题一、人口分布

### 1、世界人口分布

世界人口分布极不均衡，主要分布在：
- **纬度位置**：北半球中低纬度（20°N–60°N）
- **海陆位置**：沿海地区（距海岸 200 km 范围内）
- **地形**：平原、低地

### 2、中国人口分布

- **特点**：东南多、西北少，分布极不均衡
- **胡焕庸线**（黑河—腾冲线）：此线东南侧人口稠密，西北侧人口稀疏

### 3、影响人口分布的因素

| 自然因素 | 人文因素 |
|---|---|
| 气候（温暖湿润） | 经济发展水平 |
| 地形（平原优于山地） | 交通便利程度 |
| 水源（河流沿岸） | 历史基础 |
| 土壤（肥沃地区） | 政策因素 |

## 主题二、人口迁移

### 1、人口迁移的概念

人口在地理位置上的**长期性**和**跨越行政边界**的移动。

### 2、影响人口迁移的因素

- **推力（迁出地）**：自然灾害、就业困难、收入低、环境恶化
- **拉力（迁入地）**：就业机会多、收入高、教育医疗条件好、环境优美

### 3、人口迁移的影响

| 对迁出地 | 对迁入地 |
|---|---|
| 缓解人地矛盾 | 提供劳动力 |
| 人才流失 | 增加就业压力 |
| 增加经济收入（汇款） | 带来交通、住房、环境问题 |

## 主题三、人口容量

### 1、环境人口容量

在可预见的时期内，利用本地资源及其他资源、智力、技术等条件，在保证符合社会文化准则的物质生活水平条件下，该国家或地区所能持续供养的人口数量。

### 2、资源环境承载力

一定时期内，在维持相对稳定的前提下，资源、环境所能容纳的人口规模和经济规模的大小。

### 3、影响人口容量的因素

- **资源丰富程度**（首要因素）
- **科技发展水平**
- **消费水平**（与人口容量呈**负相关**）
- **对外开放程度**

## 主题四、城镇与乡村

### 1、聚落的概念

聚落是人类各种形式聚居地的总称，分为**城镇**和**乡村**。

### 2、城镇内部空间结构

| 功能区 | 特点 | 位置 |
|---|---|---|
| 商业区 | 占地面积小，呈点状或条状 | 市中心、交通干线两侧 |
| 住宅区 | 占地面积大，是城市最基本的功能区 | 商业区外围 |
| 工业区 | 不断向市区外缘移动 | 靠近交通干线、城市边缘 |

### 3、城市化的标志

- **城镇人口增加**
- **城镇用地规模扩大**
- **城镇人口占总人口的比例上升**（最重要的指标）

### 4、城市化进程阶段

| 阶段 | 特点 |
|---|---|
| 初期阶段 | 城市化水平低，发展缓慢 |
| 中期阶段 | 城市化加速推进，出现郊区城市化 |
| 后期阶段 | 城市化水平高（>70%），速度趋缓，出现逆城市化 |

## 主题五、农业区位因素

### 1、农业区位因素

**自然因素**：
- **气候**（热量、降水、光照）——最重要
- **地形**（平原宜耕、山地宜林牧）
- **土壤**（肥力）
- **水源**（灌溉水源）

**社会经济因素**：
- 市场（决定农业类型和规模）
- 交通运输
- 政策
- 科技（温室农业、无土栽培）
- 劳动力

### 2、主要农业地域类型

| 类型 | 分布 | 特点 |
|---|---|---|
| 季风水田农业 | 东亚、东南亚、南亚 | 精耕细作，单产高，商品率低 |
| 商品谷物农业 | 美国、加拿大、澳大利亚 | 规模大，机械化程度高，商品率高 |
| 大牧场放牧业 | 阿根廷、美国、澳大利亚 | 干旱半干旱气候区，专业化程度高 |
| 乳畜业 | 西欧、北美五大湖周围 | 市场导向型，靠近城市 |
| 混合农业 | 澳大利亚墨累—达令盆地 | 种植业与畜牧业相结合 |

## 主题六、工业区位因素

### 1、工业区位因素

| 区位因素 | 对工业的影响 |
|---|---|
| 原料 | 原料指向型（制糖、水产品加工） |
| 市场 | 市场指向型（啤酒、饮料、家具） |
| 动力（能源） | 动力指向型（有色冶金） |
| 劳动力 | 劳动力指向型（纺织、服装） |
| 技术 | 技术指向型（集成电路、精密仪表） |
| 交通 | 临海型、临路型工业布局 |
| 环境 | 污染企业布局在盛行风下风向、河流下游 |

### 2、工业集聚与工业分散

**工业集聚**：
- 优势：共享基础设施、降低运输成本、加强信息交流
- 劣势：用地紧张、环境污染加剧

**工业分散**：
- 优势：充分利用各地资源、接近市场
- 代表：跨国公司

## 主题七、大气运动

### 1、大气的受热过程

太阳短波辐射 → 地面吸收 → 地面长波辐射 → 大气吸收 → 大气逆辐射（保温作用）

**原理应用**：
- 晴天昼夜温差大（云层少，白天削弱少、夜晚保温弱）
- 阴天昼夜温差小
- 温室大棚、全球变暖

### 2、热力环流

地面受热不均 → 大气垂直运动（热上升、冷下沉）→ 同一水平面气压差 → 大气水平运动

**常见案例**：
- **海陆风**：白天风从海洋吹向陆地（海风），夜晚相反（陆风）
- **山谷风**：白天风从谷地吹向山坡（谷风），夜晚相反（山风）
- **城市热岛效应**：城市气温高于郊区，风从郊区吹向城市

### 3、气压带和风带

| 气压带/风带 | 位置 | 性质 |
|---|---|---|
| 赤道低气压带 | 赤道附近 | 湿热 |
| 信风带 | 副热带—赤道 | 干燥 |
| 副热带高气压带 | 南北纬 30° 附近 | 干热 |
| 西风带 | 副热带—副极地 | 温湿 |
| 副极地低气压带 | 南北纬 60° 附近 | 温湿 |
| 极地东风带 | 极地—副极地 | 干冷 |
| 极地高气压带 | 南北极附近 | 干冷 |

### 4、季风环流

| 季节 | 风向 | 性质 |
|---|---|---|
| 东亚冬季 | 西北风 | 寒冷干燥 |
| 东亚夏季 | 东南风 | 温暖湿润 |
| 南亚冬季 | 东北风 | 温暖干燥 |
| 南亚夏季 | 西南风 | 高温高湿 |

**季风成因**：海陆热力性质差异（主要）、气压带风带季节移动（南亚夏季风）

### 5、天气系统

| 天气系统 | 天气特征 |
|---|---|
| 冷锋 | 过境时：大风、降温、雨雪；过境后：气温下降、气压上升、转晴 |
| 暖锋 | 过境时：连续性降水；过境后：气温上升、气压下降、转晴 |
| 气旋（低压） | 中心气流上升，多阴雨天气 |
| 反气旋（高压） | 中心气流下沉，天气晴朗 |

## 主题八、海水性质与运动

### 1、海水的温度

- **水平分布**：从低纬向高纬递减
- **垂直分布**：表层到 1000 m 水温迅速下降，1000 m 以下水温变化小

### 2、海水的盐度

- **世界海洋平均盐度**：约 35‰
- **盐度分布规律**：从副热带海区向两侧递减
- **红海盐度最高**（蒸发旺盛、降水少、河流注入少）
- **波罗的海盐度最低**（蒸发弱、降水多、河流注入多）

### 3、海水的密度

- **影响因素**：温度（负相关）、盐度（正相关）
- **分布规律**：从赤道向两极递增

### 4、洋流

**按性质分**：
- **暖流**：从水温高的海区流向水温低的海区
- **寒流**：从水温低的海区流向水温高的海区

**世界洋流分布规律**：
| 海区 | 洋流方向 |
|---|---|
| 北半球中低纬 | 顺时针环流 |
| 南半球中低纬 | 逆时针环流 |
| 北半球中高纬 | 逆时针环流 |
| 南半球中高纬 | 西风漂流（自西向东） |

**洋流对地理环境的影响**：
- **气候**：暖流增温增湿，寒流降温减湿
- **渔场**：寒暖流交汇处（北海道、纽芬兰、北海）和上升补偿流（秘鲁）
- **航海**：顺流加速，逆流减速；寒暖流交汇处多海雾
- **污染**：加快净化速度，扩大污染范围

## 主题九、海洋灾害

### 1、赤潮

在特定的环境条件下，海水中某些浮游植物、原生动物或细菌爆发性增殖或高度聚集而引起水体变色的一种有害生态现象。

**成因**：
- **自然原因**：水温升高、光照充足、海水流动性差
- **人为原因**：含氮、磷等营养物质的工业废水、农业废水、生活污水大量排入海洋

**危害**：
- 水体缺氧，导致鱼类大量死亡
- 破坏海洋生态系统
- 产生毒素，危害人体健康

**防治措施**：
- 控制陆源污染，减少氮磷排放
- 加强海洋环境监测
- 合理开发海洋资源

### 2、风暴潮

由热带气旋、温带气旋等强烈大气扰动引起的海面异常升高现象。

### 3、海啸

由海底地震、火山爆发、海底滑坡等引起的具有强大破坏力的海浪。

## 主题十、主要地貌类型

### 1、流水地貌

| 类型 | 形成过程 | 典型地区 |
|---|---|---|
| **峡谷（V形谷）** | 河流上游，下切侵蚀强烈 | 长江三峡、雅鲁藏布大峡谷 |
| **冲积扇** | 河流出山口，流速减慢，泥沙堆积 | 山前地带 |
| **河漫滩** | 河流中下游，侧蚀与堆积作用 | 河流两岸 |
| **曲流（河曲）** | 河流中游，凹岸侵蚀、凸岸堆积 | 长江中下游 |
| **三角洲** | 河流入海口，流速极慢，泥沙堆积 | 长江三角洲、珠江三角洲 |

### 2、风成地貌

| 类型 | 形成过程 | 典型地区 |
|---|---|---|
| **风蚀地貌** | 风力侵蚀作用 | 风蚀蘑菇、风蚀柱、风蚀残丘 |
| **风积地貌** | 风力堆积作用 | 沙丘（新月形沙丘）、黄土高原 |

### 3、喀斯特地貌（岩溶地貌）

- **地表喀斯特**：溶沟、石芽、峰林、孤峰
- **地下喀斯特**：溶洞、石钟乳、石笋、石柱

**典型地区**：广西桂林、云南石林、贵州

**形成条件**：可溶性岩石（石灰岩）+ 温暖湿润气候 + 水与二氧化碳作用

### 4、冰川地貌

- **侵蚀地貌**：U形谷、角峰、刃脊、冰斗
- **堆积地貌**：冰碛丘陵、终碛堤

**典型地区**：青藏高原、阿尔卑斯山脉

### 5、海岸地貌

| 类型 | 特点 | 典型地区 |
|---|---|---|
| **海蚀地貌** | 海蚀崖、海蚀柱、海蚀穴 | 大连、青岛 |
| **海积地貌** | 海滩、沙坝、潟湖 | 海南、北海 |

## 主题十一、自然环境的地域差异性

### 1、水平地域分异

| 分异规律 | 主导因素 | 典型地区 |
|---|---|---|
| 从赤道到两极（纬度地带性） | 热量 | 低纬、高纬地区 |
| 从沿海到内陆（经度地带性） | 水分 | 中纬度大陆地区 |

### 2、垂直地域分异——山地垂直自然带

- **主导因素**：热量和水分随海拔变化
- 基带与山地所处的水平自然带一致
- 纬度越低、海拔越高，垂直自然带越丰富

## 主题十二、环境问题与可持续发展

### 1、主要环境问题

| 类型 | 表现 | 成因 |
|---|---|---|
| 全球变暖 | 气温上升、冰川融化、海平面上升 | CO₂ 等温室气体增多 |
| 臭氧层破坏 | 臭氧层变薄 | 氟氯烃（CFC）的使用 |
| 酸雨 | pH < 5.6 的降水 | SO₂、NOₓ 的排放 |
| 土地荒漠化 | 土地生产力下降 | 过度开垦、过度放牧 |
| 水土流失 | 土壤流失、肥力下降 | 植被破坏、不合理耕作 |
| 生物多样性减少 | 物种灭绝速度加快 | 人类活动破坏栖息地 |

### 2、可持续发展的基本内涵

**定义**：既满足当代人的需求，又不危及后代人满足其需求能力的发展。

**基本原则**：
- **公平性原则**——代内公平、代际公平、人与生物公平
- **持续性原则**——资源永续利用、生态持续发展
- **共同性原则**——全球合作，共同行动

### 3、循环经济

- **原则**：减量化（Reduce）、再利用（Reuse）、再循环（Recycle）
- **特征**：低投入、高利用、低排放
