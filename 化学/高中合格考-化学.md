## 二、甲烷的化学性质

### 1、甲烷的氧化反应 —— 燃烧反应

甲烷与氧气燃烧的化学方程式：

$$\text{CH}_4 + 2\text{O}_2 \xrightarrow{\text{点燃}} \text{CO}_2 + 2\text{H}_2\text{O}$$

### 2、甲烷的取代反应

#### ① 反应的机理

甲烷与氯气在**光照**条件下发生取代反应，逐步取代：

**第一步**：生成一氯甲烷（气态）

$$\text{CH}_4 + \text{Cl}_2 \xrightarrow{\text{光照}} \text{CH}_3\text{Cl} + \text{HCl}$$

**第二步**：生成二氯甲烷（液态）

$$\text{CH}_3\text{Cl} + \text{Cl}_2 \xrightarrow{\text{光照}} \text{CH}_2\text{Cl}_2 + \text{HCl}$$

**第三步**：生成三氯甲烷（液态）

$$\text{CH}_2\text{Cl}_2 + \text{Cl}_2 \xrightarrow{\text{光照}} \text{CHCl}_3 + \text{HCl}$$

**第四步**：生成四氯甲烷（液态）

$$\text{CHCl}_3 + \text{Cl}_2 \xrightarrow{\text{光照}} \text{CCl}_4 + \text{HCl}$$

#### ② 化学反应方程式汇总

$$\text{CH}_4 + \text{Cl}_2 \xrightarrow{\text{光照}} \text{CH}_3\text{Cl} + \text{HCl}$$

$$\text{CH}_3\text{Cl} + \text{Cl}_2 \xrightarrow{\text{光照}} \text{CH}_2\text{Cl}_2 + \text{HCl}$$

$$\text{CH}_2\text{Cl}_2 + \text{Cl}_2 \xrightarrow{\text{光照}} \text{CHCl}_3 + \text{HCl}$$

$$\text{CHCl}_3 + \text{Cl}_2 \xrightarrow{\text{光照}} \text{CCl}_4 + \text{HCl}$$

## 三、乙烯的化学性质

### 1、乙烯的氧化反应

| 实验过程 | 实验现象 |
|---|---|
| （1）点燃纯净的乙烯，观察燃烧时的现象 | 火焰明亮且伴有黑烟，同时放出大量的热 |
| （2）将乙烯通入盛有酸性高锰酸钾溶液的试管中，观察现象 | 酸性高锰酸钾溶液褪色 |

### 2、乙烯的加成反应

| 实验过程 | 实验现象 |
|---|---|
| 将乙烯通入盛有溴的四氯化碳溶液的试管中，观察现象 | 溴的四氯化碳溶液褪色 |

#### （1）乙烯与溴的四氯化碳溶液（或溴水）反应

$$\text{CH}_2\text{=CH}_2 + \text{Br}_2 \longrightarrow \text{CH}_2\text{Br—CH}_2\text{Br} \quad (1,2\text{—二溴乙烷})$$

#### （2）乙烯与其它加成试剂反应

乙烯可与 H₂、卤化氢、H₂O、卤素单质发生加成反应：

**① 与 H₂ 加成**：

$$\text{CH}_2\text{=CH}_2 + \text{H}_2 \xrightarrow[\Delta]{\text{催化剂}} \text{CH}_3\text{—CH}_3$$

**② 与 HCl 加成**（制取纯净的氯乙烷）：

$$\text{CH}_2\text{=CH}_2 + \text{HCl} \xrightarrow[\Delta]{\text{催化剂}} \text{CH}_3\text{CH}_2\text{Cl}$$

**③ 与 H₂O 加成**（制取乙醇）：

$$\text{CH}_2\text{=CH}_2 + \text{H}_2\text{O} \xrightarrow[\text{加热、加压}]{\text{催化剂}} \text{CH}_3\text{CH}_2\text{OH}$$

**④ 与 Cl₂ 加成**：

$$\text{CH}_2\text{=CH}_2 + \text{Cl}_2 \longrightarrow \text{CH}_2\text{Cl—CH}_2\text{Cl}$$

### 3、乙烯的加聚反应

**反应的方程式**：

$$n\text{CH}_2\text{=CH}_2 \xrightarrow{\text{催化剂}} \text{—[CH}_2\text{—CH}_2\text{]}_n\text{—}$$

以氯乙烯为例：

$$n\text{CH}_2\text{=CHCl} \xrightarrow{\text{催化剂}} \text{—[CH}_2\text{—CHCl]}_n\text{—} \quad (\text{聚氯乙烯})$$

- **单体**：CH₂=CHCl（含有碳碳双键的小分子）
- **链节**：—CH₂—CHCl—（重复单元）
- **聚合度**：n

## 四、乙醇

**分子式**：CH₃CH₂OH 或 C₂H₅OH

### 物理性质

乙醇俗称酒精，无色、有特殊香味的液体，密度比水的小，易挥发，与水以任意比例互溶，能溶解多种有机物和无机物。

### 化学性质

#### 1、乙醇与钠反应

$$2\text{C}_2\text{H}_5\text{OH} + 2\text{Na} \longrightarrow 2\text{CH}_3\text{CH}_2\text{ONa} + \text{H}_2\uparrow$$

#### 2、乙醇的氧化反应

**（1）乙醇的燃烧反应**：

$$\text{CH}_3\text{CH}_2\text{OH} + 3\text{O}_2 \xrightarrow{\text{点燃}} 2\text{CO}_2 + 3\text{H}_2\text{O}$$

**（2）乙醇与氧气的催化氧化**：

$$2\text{CH}_3\text{CH}_2\text{OH} + \text{O}_2 \xrightarrow[\Delta]{\text{Cu或Ag}} 2\text{CH}_3\text{CHO} + 2\text{H}_2\text{O}$$

- 实验现象：① 灼烧至红热的铜丝表面变黑，趁热将铜丝插入乙醇中，铜丝立即又变成红色；② 能闻到一股不同于乙醇的强烈的刺激性气味。

**（3）乙醇与强氧化剂反应**：

乙醇与酸性高锰酸钾溶液或酸性重铬酸钾溶液反应，被直接氧化为乙酸。

反应原理：

$$\text{CH}_3\text{CH}_2\text{OH} \xrightarrow[\text{K}_2\text{Cr}_2\text{O}_7(\text{H}^+)]{\text{KMnO}_4(\text{H}^+)} \text{CH}_3\text{COOH}$$

- **应用**：酸性重铬酸钾溶液遇乙醇后，溶液由橙色变为绿色，该反应可以用来检验司机是否酒后驾车。

#### 3、乙醇的消去反应 —— 乙烯的实验室制法

在实验室中，通常是通过加热酒精和浓硫酸的混合物，使酒精分解而制得乙烯。

- ① 实验药品：乙醇、浓硫酸
- ② 反应原理：

$$\text{CH}_3\text{CH}_2\text{OH} \xrightarrow[170°C]{\text{浓H}_2\text{SO}_4} \text{CH}_2\text{=CH}_2\uparrow + \text{H}_2\text{O}$$

## 五、乙酸

**分子式**：CH₃COOH

### 物理性质

乙酸俗名醋酸，是一种无色液体，具有强烈刺激性气味，易挥发，熔点 16.6°C，沸点 108°C，易溶于水和乙醇。当温度低于 16.6°C 时，乙酸就会凝结成像冰一样的晶体，所以无水乙酸又叫**冰醋酸**。

### 化学性质

乙酸是一种重要的有机酸，具有酸性，比 H₂CO₃ 的酸性强，在水中可以电离出 H⁺：

$$\text{CH}_3\text{COOH} \rightleftharpoons \text{CH}_3\text{COO}^- + \text{H}^+$$

是一元弱酸，具有酸的通性。

#### 乙酸的酸性与相关化学方程式

| 乙酸的酸性 | 相关的化学方程式 |
|---|---|
| ① 乙酸能使紫色石蕊溶液变红色 | — |
| ② 与活泼金属(Na)反应 | 2Na + 2CH₃COOH → 2CH₃COONa + H₂↑ |
| ③ 与某些金属氧化物(CaO)反应 | CaO + 2CH₃COOH → (CH₃COO)₂Ca + H₂O |
| ④ 与碱[NaOH、Cu(OH)₂]发生中和反应 | CH₃COOH + NaOH → CH₃COONa + H₂O；Cu(OH)₂ + 2CH₃COOH → (CH₃COO)₂Cu + 2H₂O |
| ⑤ 与某些盐(Na₂CO₃、NaHCO₃)反应 | Na₂CO₃ + 2CH₃COOH → 2CH₃COONa + CO₂↑ + H₂O；CH₃COOH + NaHCO₃ → CH₃COONa + CO₂↑ + H₂O |

### 反应方程式

$$\text{CH}_3\text{—C(=O)—OH} + \text{H—O—C}_2\text{H}_5 \xrightleftharpoons[\Delta]{\text{浓H}_2\text{SO}_4} \text{CH}_3\text{—C(=O)—O—C}_2\text{H}_5 + \text{H}_2\text{O}$$

## 七、同系物

### 1、定义

结构相似，在分子组成上相差一个或若干个 CH₂ 原子团的物质，互称为同系物。

如：CH₄、C₂H₆、C₃H₈ 互为同系物。

### 2、判断方法

- ① **一差**：两种物质分子组成上相差一个或多个 "CH₂" 原子团
- ② **二同**：具有相同的通式，两种物质属于同一类物质
- ③ **三相似**：两种物质结构相似，化学性质相似

## 九、有机物组成与结构的表示方法

| 种类 | 实例 | 含义 | 应用范围 |
|---|---|---|---|
| 分子式 | C₂H₆ | 用元素符号表示物质组成的式子，可以反映出一个分子中原子的种类和数目 | 多用于研究分子晶体 |
| 最简式（实验式） | CH₃ | 表示组成物质的各元素原子最简整数比的式子；由最简式可求最简式量 | 有共同组成的物质 |
| 电子式 | H:C:C:H（上下各两个H） | 用小黑点 "·" 或 "×" 代表电子，表示原子最外层电子成键情况的式子 | 多用于表示离子型、共价型的物质 |
| 结构式 | H—C(—H)(—H)—C(—H)(—H)—H | 用一根短线表示一对共用电子对的式子。具有化学式所能表示的意义，能反映物质的结构；表示分子中原子的结合或排列顺序的式子，但不能表示空间构型 | 多用于研究有机物的性质；由于能反映有机物的结构，有机物常用结构式表示 |
| 结构简式 | CH₃—CH₃ | 省去结构式中代表碳氢单键、碳碳单键的短线，即得到结构简式。结构简式的简便写法，着重突出结构的特点 | |
| 键线式 | CH₃CH=CHCH(CH₃)CH₃ 可表示为折线图 | 将结构式中的碳、氢元素符号省去，每个拐点、终点均表示一个碳原子 | 用于表示分子中的键的连接情况 |
| 碳架式 | C—C | 省去结构式中所有氢原子，只写出碳原子和共价键 | |
| 球棍模型 | 小球+棍的立体模型 | 小球表示原子，短线表示共价键 | 用于表示分子的空间结构（立体结构） |
| 空间充填模型 | 密堆积球体模型 | 用不同体积的小球表示不同大小的原子 | 用于表示分子中各原子的相对大小和综合顺序 |
